Guide to the Web Interface

This page provides a guide to the features of the web interface.

The Index

The work index page contains a list of all of the works on the system and their associated 'material' (texts etc).

Multiviews

You can view mutiple texts at once by concatenating the name-ids separated by '+'. For example to see the Gutenberg folio and non-folio Othello side-by-side you would visit: ${h.url_for(controller='multi', action='view', id=None, text_1='othello_gut_f', text_2='othello_gut')}

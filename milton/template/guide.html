Guide to the Web Interface

This page provides a guide to the features of the Open Milton web interface.

The Index

The index page contains a list of all of the material on the system.

Viewing Works

You can view works by clicking on the link available from the index page. If you know the 'id' of a text you can view it directly by visiting the url: /view?name={id} (you can work out id names by looking at the link urls on the index page).

Viewing in different formats

You can view a given text in a different format by setting the format argument in the url query string like so: /view?name={id}&format={format_name} So to view the raw Poemata text with line numbers visit: /view?name=poemata_gut&format=raw Available formats are:

  • format=plain - plain version of the file (default)
  • format=raw - raw file returned as text/plain

Concordance

The concordance page provides a concordance of Milton's work along with some associated statistics on word occurences.
